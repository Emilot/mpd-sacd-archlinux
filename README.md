# mpd-sacd-archlinux
A patch for mpd-sacd@aur.archlinux.org
---
## Installation
1. Get all the files
```bash
    git clone https://github.com/bruce-awareit/mpd-sacd-archlinux.git
```
2. Change to the folder
```bash
    cd mpd-sacd-archlinux
```
3. Make and Install the package
```bash
    makepkg -i
```
